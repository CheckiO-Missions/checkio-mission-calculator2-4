在本系列任务中，您将制作一个初级计算器。

作为输入，您需要按下一连串的按键，而作为该功能的结果，您应该显示出按下最后一个键时屏幕上将显示的内容。注意，这并不总是表达式的结果。实际上，使用实物计算器或应用程序确实有助于你捕捉边缘情况。

在第四个任务中，允许使用浮点数。. "占用一位数的空间，因此浮点数结果的最大位数空间为 4。这意味着您可以输入的最大浮点数是 "9999."。

预期行为

  • 应删除开头的零，仅有零的数字 - 转换为单个零；
  • 数字之间的 +- 符号应取最后一个；
  • "=="表示重复上次操作；
  • "+="或"-="--在组合前对数字（或运算结果）进行加减运算（将数字加倍或减去本身）；
  • 计算器忽略在第 5 位之后输入的数字；
  • 数字 < 0 的"-"不占位数；
  • 如果 abs(结果)大于 99999，结果显示为 "错误"；
  • 对于浮点数，如果 abs(结果) 的整数部分大于 9999，则结果显示为 "错误"；
  • 对于 float，如果数字的总长度超过 5 位数，则应四舍五入到 5 位数（1.23456 -> 1.235）；
  • 对于浮点数，应去掉开头和尾部的零（如果可能，去掉". "之前的零）： 0.1200 -> .12 , 123.00 -> 123..应在四舍五入后进行： 1.000123 -> 1. .只有在输入数字结束后（按下非数字字符）才应删除尾数零。
